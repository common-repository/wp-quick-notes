Allow users to write down notes on frontend.

== Description ==
This simple plugin allow users to write down notes and print them later. This plugin will be useful if you are running a podcast web site.

Features

1. WYSIWYG front-end editor.
2. Print button
3. No setup required

== Installation ==
1. Upload "quick-notes.zip" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= How to display the quick notes widget? =
Drag and drop the quick notes widget into the desired sidebar.
